## BMR - Bangalore Malayali Riders

### Objective
Build a centralized platform for managing BMR activities such as member and event management. Primary users of this product would be BMR members.


### Techstack
#### Frontend
- NextJS 14
- Axios
- Tailwind
- Typescript
- Formik
- Yup

#### Backend
- NodeJS
- MongoDB
- Joi
- Typescript
- JWT
